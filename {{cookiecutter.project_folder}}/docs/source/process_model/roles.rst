Roles
=====

Roles represent an abstraction of knowledge and skills, which are needed for the development of an application in the NLP context. They do not embody certain individuals. 
One role is not limited to one person. And one person can have several roles.

The following roles are part of the NLP Process Model and thus this roles are needed to process the model.
